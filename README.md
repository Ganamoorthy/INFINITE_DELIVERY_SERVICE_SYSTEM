# Infinite Delivery Service System

## Overview
Infinite Delivery Service System is a Java-based application designed to streamline online ordering and delivery management for small and medium-scale delivery services. It offers features for secure customer login, order placement, courier selection, real-time delivery charge calculation, and multiple payment options.

## Features
- Secure customer management with login and profile details
- Order placement with item details, quantity, and delivery preferences
- Courier selection from a predefined list with detailed contact information
- Automated delivery charge calculation based on weight, distance, and rate tables
- Flexible payment methods: Cash on Delivery and Online Payment
- Real-time order processing and charge estimation
- User-friendly interface for customers and service providers

## Project Structure
- `Main.java`: Program entry point and main application logic
- `Customer.java`: Handles customer data and authentication
- `User.java`: General user management superclass or related class
- `Order.java`: Order creation and management
- `Courier.java`: Courier information and selection management
- `Charge.java`: Delivery charge calculation module
- `PaymentMethod.java`: Payment processing and method definitions

## Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or later
- An IDE such as IntelliJ IDEA, Eclipse, or NetBeans (optional but recommended)

### Installation
1. Clone the repository:
2. Import the project into your Java IDE.
3. Build the project to resolve dependencies and compile the code.

### Running the Application
Run the `Main` class. The application will launch and you can interact with the system through the provided user interface.

## Usage
- Register or login as a customer
- Place an order specifying item details and delivery options
- Select a courier from the list provided
- Review the calculated delivery charges
- Select a payment method and confirm the order
